# Weather Application (weather-parent)

Dieses Projekt ist eine einfache Java-Anwendung zur Simulation und Abfrage von Wetterdaten.

## Technologien

- Java 17  
- Maven  
- JUnit 5  
- Mockito  
- Log4J  
- SpotBugs, PMD, Checkstyle  
- Getestet und entwickelt mit **Visual Studio Code**
